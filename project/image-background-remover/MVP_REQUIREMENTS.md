## 一、产品概述

### 1.1 产品定位
一款基于 Remove.bg API 的在线图片背景移除工具，通过 Cloudflare Pages 部署，提供零服务器成本、即开即用的轻量化 SaaS 服务。

### 1.2 目标用户
- 电商卖家（产品图去底）
- 设计师（快速抠图）
- 普通用户（证件照、生活照处理）
- 开发者（API 集成测试）

### 1.3 核心价值
| 价值点 | 说明 |
|--------|------|
| **零成本启动** | Cloudflare Pages 免费托管，无服务器成本 |
| **即开即用** | 无需注册登录，打开网页即可使用 |
| **高质量输出** | 基于 Remove.bg 专业 AI 模型 |
| **隐私保护** | 图片内存处理，不落盘存储 |

## 五、API 设计

### 5.1 Remove.bg API 调用

**端点：** `https://api.remove.bg/v1.0/removebg`

**请求：**
```javascript
const formData = new FormData();
formData.append('image_file', fileInput.files[0]);
formData.append('size', 'auto');

const response = await fetch('https://api.remove.bg/v1.0/removebg', {
  method: 'POST',
  headers: {
    'X-Api-Key': apiKey
  },
  body: formData
});
```

**响应：**
- 成功：`200 OK` - 返回 PNG 图片 Blob
- 失败：`4xx/5xx` - 返回 JSON 错误信息

### 5.2 错误码处理

| 状态码 | 含义 | 用户提示 |
|--------|------|----------|
| 400 | 请求参数错误 | "图片格式不支持，请使用 JPG/PNG/WebP" |
| 401 | API Key 无效 | "API Key 无效，请检查配置" |
| 402 | 余额不足 | "API 配额已用尽，请充值或更换 Key" |
| 403 | 图片过大 | "图片超过 5MB，请压缩后重试" |
| 429 | 请求限流 | "请求过于频繁，请稍后再试" |
| 500 | 服务器错误 | "服务暂时不可用，请稍后重试" |

## 七、安全与隐私

## 八、部署方案

### 8.1 Cloudflare Pages 部署

**步骤：**
1. 创建 GitHub 仓库
2. 登录 Cloudflare Dashboard
3. 创建 Pages 项目 → 连接 GitHub 仓库
4. 配置构建设置（无需构建，纯静态）
5. 添加环境变量（可选）：`REMOVEBG_API_KEY`
6. 部署完成，获取域名

**域名：**
- 默认：`xxx.pages.dev`
- 自定义：绑定自有域名

### 8.2 环境变量配置

| 变量名 | 说明 | 是否必须 |
|--------|------|----------|
| `REMOVEBG_API_KEY` | 默认 API Key（可选） | 否 |
| `SITE_TITLE` | 网站标题 | 否 |
| `MAX_FILE_SIZE` | 最大文件大小（字节） | 否 |
